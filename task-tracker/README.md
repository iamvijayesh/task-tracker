# Task Tracker

A simple and intuitive **Task Tracker web application** built as part of the **Full-Stack Internship assignment for NOVA Technology**.

The project focuses on clean frontend implementation, basic state management, and clarity of logic — exactly as required by the assignment.

## What This App Does (at a glance)

- Add a new task  
- View all tasks in a list  
- Mark tasks as **Completed** or **Pending**  
- Tasks persist even after page refresh  
-  Clean, minimal, and user-friendly UI  

## Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (ES6)** – Logic & interactivity
